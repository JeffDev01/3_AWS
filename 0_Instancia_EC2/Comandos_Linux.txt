#EDITANDO PERMISSÕES

chwn :define quem é o proprietário do arquivo (User, Group e Other) permissões: r(read) w(write) x(execution)

#EDITAR TEXTO NO LINUX
Abra um arquivo digitando:
vi <nome do arquivo>
Aperte a tecla INSERT ou i para editar o arquivo
Edite o texto, use as setas para mover o cursor
Ao terminar a edição aperte a tecla ESC
Digite :w para salvar as modificações, :wq para salvar e sair do vi e :q! para sair sem salvar.
Para criar um arquivo digite
vi <nome do arquivo que deseja criar>
Proceda como no caso da edição de um arquivo.
